Open Megavideo videos Full Screen in a New Window. Just use [mega]paste Megavideo url here[/mega] and it will create a link to open the Megavideo video full screen in a new window, in the Megavideo player.

== Description ==

**** Open Megavideo Full Screen in a New Window. Just use [mega]paste Megavideo url here[/mega] and it will create a link to open the Megavideo video full screen in a new window, in the Megavideo player.

For more information visit the plugin website. <a href="http://megavideopop.com" target="_blank" title="MegaVideoPop">MegaVideoPop.com</a>

#### **Example code:**

**This:** Click here to [mega]http://www.megavideo.com/?v=JPN400BH[/mega]
**Becomes this:** Click here to <a href="http://wwwstatic.megavideo.com/mv_player.swf?v=JPN400BH" target="_blank">Watch Now</a>


== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
